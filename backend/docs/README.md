# docs/ - 文档目录

此目录存放项目文档。

## 建议内容

- API 文档
- 架构设计文档
- 部署文档
- 开发指南

## 目录结构建议

```
docs/
├── api/            # API 文档
│   └── swagger.yaml
├── architecture/   # 架构文档
├── deployment/     # 部署文档
└── development/    # 开发指南
```

## 文档类型

### API 文档
- OpenAPI/Swagger 规范
- 接口说明和示例

### 架构文档
- 系统架构图
- 技术选型说明
- 数据库设计

### 部署文档
- 环境要求
- 部署步骤
- 配置说明

### 开发指南
- 本地开发环境搭建
- 代码规范
- Git 工作流

## 工具推荐

- Swagger UI - API 文档展示
- Draw.io - 架构图绘制
- MkDocs - 文档站点生成
