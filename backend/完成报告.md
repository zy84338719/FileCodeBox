## 🚀 启动方式

### 后端

```bash
cd /opt/project/claude/FileCodeBox_cloudwego

# 首次运行
go mod tidy

# 编译
go build -o filecodebox

# 运行
./filecodebox

# 访问
http://localhost:8888
```

### 前端

```bash
cd /opt/project/claude/FileCodeBox-frontend

# 安装依赖
npm install

# 开发模式
npm run dev

# 访问
http://localhost:3000
```
